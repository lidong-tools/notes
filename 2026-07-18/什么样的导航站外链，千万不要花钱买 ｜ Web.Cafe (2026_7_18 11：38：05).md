# 什么样的导航站外链，千万不要花钱买

本文内容基本上是根据哥飞、论坛上一些其他群友的分享，再结合一些自己的实操经验总结来的。

当然实际情况是，人在河边走哪有不湿鞋，该交的学费少不了，尽可能的情况下，少花冤枉钱就好。

# 一、可以直接过滤的

1. ahrefs标记为spam，不要花钱买。但这个免费工具看到的外链不多，怎么办呢，可以看自己的网站，在ahrefs验证过所有权的网站可以看全部的外链，有spam标记的，拉黑就好。万一不小心花钱买了一条spam😭可以安慰一下自己，这个标记是动态的，说不定过几个月就没了呢。但要注意：没有spam也不代表是好外链，best link才是真的好。有些新导航站，会主动去各大导航站抓网站，它们会主动收录给你外链，然后你会在ahrefs看到它们的spam标记，这种可以直接入库拉黑，省的下次遇到还花时间去调查，至少隔半年再从黑名单里放出来调查一遍。
2. 太便宜的不要买。虽然到处都是「5刀1000条外链千万别买」的传说，但实际上有些“太便宜的”也没这么便宜。如果要找个标准，那就是 $9以下的都谨慎一些，比如$ 1/条的就明显属于要过滤掉的。

# 二、看数据过滤的

从别人网站那里扒出来的外链，到底质量怎么样？ahrefs免费版比较难查，但也有几个可以参考的数值过滤标准。以下数据有些是有来源的，有些是我拍脑袋定的（看筛选情况主观调整）：

1. 月流量不足1000的，直接过滤⬅️有群友分享过，哥飞也提过这个数据
2. 搜索流量很少的，不足100的，直接过滤⬅️我拍脑袋定的，偏少了，但是作为筛选条件，可以主观调整。
3. 搜索流量占比低于10%的，直接过滤⬅️我拍脑袋定的，偏少了，实际30-40%+比较理想。哥飞说过，直接打开占50%左右也算正常，剩下的最大的流量来源应该是搜索。
4. 带搜索量的词，如果只有网站品牌词，就直接过滤掉。而除了网站品牌词之外，还有别的词（别的产品词、需求词、产品对比词等），而且有一定的量，是还可以的。
5. 流量暴跌过——3个月甚至1个月内，暴跌过半，甚至暴跌过80%，直接过滤。这是被google惩罚了，搜索算法更新，批量低质量页面会不给流量。很多导航站都有这种情况。⬅️有群友分享过，哥飞也提过这个数据
6. 导航站的dr就不高，比如不到10，直接过滤。实际上20+甚至30+会好一点。这个不解释，相比起其他数据来dr太好刷了，如果一个导航站连dr都刷不上去……
7. 出站链接太多——哥飞说过这一条，但我目前没怎么看过这个数据，也没有数据指标评估，如果有群友有经验，欢迎补充～

# 三、数据造假的过滤

最近发现不少导航站（比如下图这个，刷的太不用心了），有刷流量嫌疑，具体还可以看这些数据：

![image.png](https://ima-knowledge.oss-cn-shanghai.aliyuncs.com/ima-file/2026-07-18/%E4%BB%80%E4%B9%88%E6%A0%B7%E7%9A%84%E5%AF%BC%E8%88%AA%E7%AB%99%E5%A4%96%E9%93%BE%EF%BC%8C%E5%8D%83%E4%B8%87%E4%B8%8D%E8%A6%81%E8%8A%B1%E9%92%B1%E4%B9%B0%20%EF%BD%9C%20Web.Cafe%20(2026_7_18%2011%EF%BC%9A38%EF%BC%9A05).assets/image-008.png)

1. 网站平均停留时间——不足30秒⬅️我拍脑袋定的，偏少了，你想想一个正经导航站，半分钟能看什么？最近看到一些网站，之前的流量是0，最近1个月就过千甚至几千了，流量这么好搞吗？一看停留时长0😯——从此看导航站多了几个筛选条件。
2. 网站跳出率——只看一个页面且无交互就关掉的占比。越低越好——高于50%的可以打个问号⬅️我拍脑袋定的，按照哥飞的数据是30-40%甚至20%比较好，不过这个数据是指工具站/内容站，不是指导航站的，导航站可能本身会偏高一些。
3. 平均浏览页面——每个用户访问了本网站的几个页面，越多越好——低于1.5的可以怀疑一下⬅️我拍脑袋定的，偏少，2以上更好。
4. 提到刷数据，其实还有一种，导航站会给站长发邀请提交的邮件，他们给的链接是一个google搜索他们网站域名的链接，比如（以toolify为例）： https://www.google.com/search?q=toolify.ai 。又学到了骚操作……

# 四、不给外链，给跳转“假链接”的过滤

可以看看 dang.ai 给的链接，看了不少链接全是这种情况。只有 prompted 栏的赞助位给的是外链，而且是 dofollow 外链，其他的不管是黄标还是紫标，都是 3xx 跳转，基本不给权重，比 nofollow 还不如。不过 dang.ai 的产品页面会给 google 索引（google 搜 site:dang.ai/tool/产品页，能搜到这个页面）⬅️考虑到 dang.ai 这个站真的很大，流量也不少，所以进一步评估一下：如果这个listing不能带真实用户，那就是纯坑了，但如果能带真实用户，那就还行——必须写好产品介绍。

具体表现：

1. 给 3xx 跳转—— 301 勉强能传递一些权重， 302/307 不传递权重）
2. 给 js 跳转——纯坑。

# 五、给 noindex 或者页面质量太差 google 不索引的，看情况过滤

1. 目前几乎没有发现给 noindex 的站

1. 页面质量太差 google 不索引的看到不少——有些是站的问题，看这个站很多产品页面，都launch2、3周了，页面还没收录，那这个站不太行。但有些是自己提交的时候的问题——尽量不要一套内容发 n 个站。你想想你都辛苦发外链了，结果为了省事发完没效果，那不是白干了/白花钱了😭来都来了，不如让AI帮你改改内容啊。

# 六、性价比——量力而行，主观决定

1. 站还不错，免费给dofollow，要求排队、挂badge回链——新手期，排就排、挂就挂，没事，大不了赚钱了再花钱去掉badge。
2. 站还不错，免费只有前3给dofollow，要求排队、给回链——忍了。但如果不回链只是影响dofollow，那不回也行，看个人选择。
3. 站一般甚至比较差，免费要排队1年、不给dofollow、要求回链——下一个站更好。
4. 站一般甚至比较差，还没有免费，付费还要\$49甚至更贵——下一个站更好。
